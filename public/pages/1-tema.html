Funciones con parámetros

1. Funciones con parámetros

Las funciones son pequeños módulos de un programa que se componen de diferentes líneas de código y están encargadas de ejecutar una tarea específica dentro de la solución informática.

La declaración de una función está compuesta por varios elementos tales como: tipo, nombre y parámetros. El tipo se trata del tipo de dato que la función va a retornar, el nombre es el identificador con que se conocerá la función dentro del programa y los parámetros son algunas de las variables que se necesitan para ejecutar la función.

Internamente las funciones pueden manejar otras variables que serían variables locales, estas se declaran dentro de las líneas de código que componen la función de acuerdo con los requerimientos del programador.

En el proceso de utilización de funciones, es necesario generar el prototipo de las mismas dentro del programa y se recomienda hacerlo posterior a las librerías. La sintaxis correcta para declarar dichos prototipos es la siguiente:

Tipo: se trata del tipo de función y debe coincidir con el tipo dato que esta va a retornar.

Nombre: consiste en el identificador por medio del cual la función será reconocida.

TipoP: es el tipo de parámetro que la función recibirá, para cada parámetro enviado debe indicarse el respectivo tipo.

parametro1: consiste en el nombre dado a cada uno de los parámetros.

Cada línea utilizada como el prototipo de una función debe terminar con el signo punto y coma (;).
